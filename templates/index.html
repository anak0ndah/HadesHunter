What is HadesHunter?

HadesHunter is a security tool designed to detect secrets, credentials, and sensitive data leaked in Microsoft 365 environments. It scans Teams conversations, Outlook emails, and OneDrive/SharePoint files for potential security risks.

Inspired by: GraphRunner, GraphSpy, and TruffleHog.

Teams Scanner

Scan all Teams conversations and chats for leaked credentials and secrets.

Email Scanner

Analyze inbox emails for passwords, API keys, and sensitive information.

File Scanner

Scan OneDrive and SharePoint files for hardcoded secrets and credentials.

  • AWS Access Keys & Secret Keys
  • Azure Client Secrets
  • Azure Connection Strings
  • Google API Keys
API & Tokens
  • GitHub Tokens (ghp_, gho_, etc.)
  • Slack Tokens & Webhooks
  • Discord Tokens
  • OpenAI API Keys
  • Stripe, SendGrid, Twilio Keys

Credentials
  • Password patterns (EN, FR, ES, DE, PL, PT, IT, NL)
  • Email:Password combinations
  • Database URLs & Connection Strings
  • Private Keys (RSA, SSH, PGP)
  • Generic API Keys & Secrets

How to Use

1. Get a Refresh Token

Use tools like roadrecon, TokenTacticsV2, or device code phishing to obtain a Microsoft refresh token.

2. Choose Your Scanner

Terminator (recommended): Scans Teams, Emails, and Files in one go.
Or use individual scanners for specific sources.

3. Configure & Scan

Paste your refresh token, select the number of items to scan, and click Scan.

4. Review Results

View detected secrets with context, confidence scores, and entropy analysis. Export or delete results as needed.
