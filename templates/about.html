About HadesHunter

What is HadesHunter?

HadesHunter is a security tool designed to scan Microsoft Teams conversations for accidentally leaked secrets, credentials, and sensitive information.

It helps security teams and penetration testers identify potential data leaks in corporate communications, such as:

  • API Keys - AWS, Azure, Google Cloud, GitHub, Slack, etc.
  • Passwords - Shared credentials in chat messages
  • Tokens - JWT, OAuth, Bearer tokens
  • Connection Strings - Database and service connections
  • Private Keys - SSH, PGP, certificates
Supported Languages

The scanner detects secrets shared in conversations in multiple languages:

  • English
  • French
  • German
  • Spanish
  • Portuguese
  • Italian
  • Dutch
  • Polish
How It Works
  1. Authentication - Provide a refresh token or access token to authenticate with Microsoft Teams API.
  2. Conversation Retrieval - The tool fetches all accessible Teams conversations and messages.
  3. Pattern Matching - Each message is scanned using regex patterns for known secret formats (API keys, tokens, etc.).
  4. Entropy Analysis - Shannon entropy is calculated to filter out false positives and identify high-randomness strings.
  5. Context Extraction - For each detected secret, surrounding messages are captured for context.
Detection Types
  • AWS Keys High
  • Azure Secrets High
  • GitHub Tokens High
  • Slack Tokens Medium
  • JWT Tokens Medium
  • Passwords Medium
  • Private Keys High
  • Connection Strings Medium
